添加设备
1. 选择接入模式 2. 填写协议参数 3. 确认入库
先添加，再接入。被动设备入库后为「待上线」，仅当设备携带正确编号/鉴权向平台注册时才变在线； 主动设备保存后由平台发起探测/拉通道。

被动 设备找平台

设备配置平台地址与自身 ID，主动向平台注册。
本期协议：国标 GB28181（后续 ISUP 等同模式扩展）。

主动 平台找设备

平台持有设备固定 IP，主动连接并拉取通道。
本期协议：ONVIF（后续私有 TCP/SDK 同模式扩展）。

被动模式下仅可选已支持的被动协议。
可选，便于后续能力差异与运维检索。
1. 选择接入模式 2. 国标参数 3. 确认入库
请先在「接入信息」中确认平台 SIP ID / IP / 端口 / 域，再把相同信息配置到摄像机； 此处填写的设备编号必须与摄像机国标 ID 一致，否则注册会被拒绝。
1. 选择接入模式 2. ONVIF 参数 3. 确认入库
保存时可勾选「校验连通性」：探测成功再入库并同步 Profile 为通道。

打开方式：用浏览器直接打开本文件。交互：顶部切换页面稿；列表点「添加设备」体验向导（模式 → 协议表单）。 设计对齐：主色 #1565c0、页面底 #f0f4f8、双列侧栏 84+176。 配套文档：docs/design/device-access/01-overview.md、02-detailed.md。
